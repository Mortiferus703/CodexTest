# Kingdom Clash

Kingdom Clash is an original browser strategy prototype inspired by village-building raid games.

## What changed in this version
- Editable home-base grid with placeable defenses, walls, and economy buildings.
- Real-time raid loop where troops are deployed from map edges and automatically path toward targets.
- Multiple troop types, heal spells, passive resource income, trophies, and enemy base scouting.

## Run locally
Open `index.html` in a browser, or serve the repository root:

```bash
python3 -m http.server 8000
```
